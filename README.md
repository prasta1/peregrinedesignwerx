# PDW Github Landing Page

A landing page for Precision Woodcraft, a CNC woodworking business.

## Overview

Simple, professional single-page website showcasing CNC machining services for custom woodworking projects.

## Features

- **Responsive Design**: Mobile-friendly layout
- **Service Overview**: Highlights three main service categories:
  - Custom Signage
  - Production Parts
  - Prototyping
- **Navigation**: Smooth scrolling to sections (Services, Gallery, Contact)
- **Contact Section**: Email contact information

## Tech Stack

- HTML5
- CSS3 (custom styles in `style.css`)
- Pure static site (no frameworks or build process)

## Sections

1. **Hero**: Bold tagline "Crafted by Code. Finished by Hand."
2. **Services**: Grid layout showcasing CNC capabilities
3. **Contact**: Simple email contact

## Usage

Open `index.html` in a web browser or deploy to any static hosting service (GitHub Pages, Netlify, Vercel, etc.).

## Customization

- Update branding in `<nav>` section
- Modify service descriptions in the `.services` section
- Replace contact email in the `#contact` section
- Adjust colors and styling in `style.css`
